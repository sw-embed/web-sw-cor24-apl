name: Demos alphabetical order
description: Demo programs in the dropdown should be listed in alphabetical order by name, not pedagogical order
type: feedback

Demos should be listed in alphabetical order by name in the demo registry/dropdown.
A tutorial (future feature) could introduce concepts in easy-to-hard order, but the demo list itself is alphabetical.

**Why:** User preference for predictable, scannable ordering in the UI.
**How to apply:** When adding new demos to src/demos.rs, insert them in alphabetical position by name.
