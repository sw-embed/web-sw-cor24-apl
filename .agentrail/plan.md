Build a browser-based Yew/Rust/WASM APL environment for COR24. Single-panel REPL with printer-style emulation, floating hardware panel (S2 switch, D2 LED, UART TX/RX), demo dropdown, file upload, and reset. Phase 1: scaffold Yew project with Trunk, wire cor24-emulator path dep. Phase 2: REPL panel (scrolling output, 6-space indent prompt, keyboard capture) + control bar. Phase 3: emulator integration (EmulatorBridge, UART I/O bridge, batch execution, pre-assemble APL interpreter via build.rs). Phase 4: hardware panel + demo registry + file upload. Phase 5: polish, responsive design, GitHub Pages deployment. Phase 6: APL glyph prettification display modes. Phase 7: New demos showcasing interpreter features (bracket indexing, bitwise ops, reverse/catenate, take/drop, error handling, control flow loops). Phase 8: Extend prettification to cover new keywords (rev, cat, and, or, not, goto). Phase 9: Command history with up/down arrow keys. Phase 10: Scroll-lock behavior (auto-scroll pauses when user scrolls up). Phase 11: APL quick-reference help overlay. Phase 12: Deploy all new features to GitHub Pages.
