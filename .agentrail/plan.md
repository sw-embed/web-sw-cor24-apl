Build a browser-based Yew/Rust/WASM APL environment for COR24. Single-panel REPL with printer-style emulation, floating hardware panel (S2 switch, D2 LED, UART TX/RX), demo dropdown, file upload, and reset. Phase 1: scaffold Yew project with Trunk, wire cor24-emulator path dep. Phase 2: REPL panel (scrolling output, 6-space indent prompt, keyboard capture) + control bar. Phase 3: emulator integration (EmulatorBridge, UART I/O bridge, batch execution, pre-assemble APL interpreter via build.rs). Phase 4: hardware panel + demo registry + file upload. Phase 5: polish, responsive design, GitHub Pages deployment. Phase 6: APL glyph prettification display modes.
